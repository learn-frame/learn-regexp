# 位置匹配

位置(也称为锚)是指每个字符之间的"空隙", 如下图箭头所指即为位置.

![位置](../images/位置.jpg)

在 ES5 中有 6 种锚, 具体如下.

| 锚  | 含义                              |
| :-: | :-------------------------------- |
|  ^  | 匹配开头, 在多行匹配中匹配行开头. |
| \$  | 匹配结尾, 在多行匹配中匹配行结尾. |
| ??  | ?                                 |
| +?  | +                                 |
| \*? | \*                                |
